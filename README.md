Live Validation Form
A modern and responsive form validation project built with **HTML, CSS, and JavaScript**. The form validates user input in real time, providing instant feedback and improving the overall user experience
 **Features**
- Real-time input validation
- Name validation
- Email validation
- Password strength validation
- Confirm password matching
- Error messages for invalid inputs
- Responsive design
- Clean and user-friendly interface
 **Technologies Used**
- HTML5
- CSS3
- JavaScript (ES6)
